# dotfiles

Matrix green-on-black Hyprland rice.

## Install

```bash
git clone https://github.com/krypton-0x00/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Stack

| Component | Choice |
|-----------|--------|
| WM | Hyprland |
| Terminal | Kitty |
| Bar | Waybar |
| Launcher | Rofi |
| Notifications | SwayNC |
| Notif Daemon | Dunst |
| Shell Prompt | Starship |
| Lock Screen | Hyprlock |
| Idle | Hypridle |
| Wallpaper | Hyprpaper |
| GTK Theme | trinity-gtk-theme |
| Font | Hack Nerd Font |
